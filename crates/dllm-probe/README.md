# dllm-probe

Standalone diagnostic tool for testing DLLM peer transport.

## Binary

Installed as `dllm-dev-probe`.

## Commands

- `id` - Generate peer ID from seed
- `forwarder` - Run a forwarding node
- `discover` - Discover and connect to providers
- `discover-listen` / `discover-listen-any` - Listen for discovered peers
- `discover-dial` - Dial a discovered peer
- `listen` / `dial` - Test relay connectivity
